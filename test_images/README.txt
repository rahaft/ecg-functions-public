ECG Test Images Folder
======================

Place ECG images (.png, .jpg) in this folder for local testing.

To get test images:
1. Download from Kaggle competition: https://kaggle.com/competitions/physionet-ecg-image-digitization/data
2. Or copy images from your GCS bucket manually
3. Or use any ECG images you have

The script will process all images in this folder when running in LOCAL mode.

Output will be saved to: ..\output_v7\
